# INCとは
**INC（Intermedia Network Center）** は、VRChat上に常設されている日本語話者向けの集会場ワールドです。

INCは、学術・技術・芸術・音楽など、様々な分野に関心を持つ方々が集まり、自発的な交流が可能な集会場となることを目指しています。

![INCメインホール](/inc/assets/images/content/docs/スクリーンショット 2026-04-29 001303.png)

## 空間について

現実の建築思想からインスピレーションを受けた、落ち着きのある空間です。室内はすべてつながったオープンな構造であり、仕切られた部屋を持たない開放的な設計となっています。

庭・メインホール・地下ギャラリー・DJエリアなど、多様な用途に対応したエリアが一体となって構成されています。

## 運営について

INCは、運営の銀こんにゃくとINC製作チームによって制作・運営されています。
